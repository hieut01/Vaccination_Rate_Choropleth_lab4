# Washington State COVID-19 Map

This project shows COVID-19 case data across Washington State using Mapbox GL JS. The map displays confirmed cases by county as of October 25, 2021.

## Files
- index.html for the Washington COVID-19 map
- pop_density.html for the US population density map
- assets folder with state_data.geojson and wa-covid-data-102521.geojson

## Features
- Choropleth map of COVID-19 cases
- Hover to show county data
- Legend explaining color scale
- Proper map title and centered on Washington

## How to View
Open index.html in a web browser to view the map.
